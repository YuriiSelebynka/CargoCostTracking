Information about product

This software product is created to account a cargo delivering in transportation companies. Program consists of:
1. security menu (login and password are requested);
2. main menu - includes buttons to access to tables and info page;
3. Routes table - includes data about routes and menus for creating new rout and for updating current;
4. Chauffeur table - includes data about chauffeurs and menus for registration new chauffeur and for updating current;
5. Finished work table - includes data about finished works and menus for registration new finished work and for updating current.
